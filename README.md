🗳️ Voting App
📌 Overview

This project is a Modern JavaScript App built with React for the frontend and Node.js for the backend. It provides hands-on experience in:

    RESTful API development
    External service integration
    State management in React
    MongoDB schema design
    Responsive UI development

🛠️ Tech Stack
Frontend

    React
    Vite
    TypeScript
    CSS/Styled Components

Backend

    Node.js
    Express.js
    MongoDB & Mongoose
    dotenv (for environment variables)
    CORS
